The Code

<div id="count" class="counter-display">0</div> <button _="on click decrement #count"> ‚ûñ Decrease </button> <button _="on click put 0 into #count"> üîÑ Reset </button> <button _="on click increment #count"> ‚ûï Increase </button>

How it works:

  • increment #count - Adds 1 to the current value
  • decrement #count - Subtracts 1 from the current value
  • put 0 into #count - Resets to zero

Advanced Increment/Decrement:

  • increment #count by 5 - Add 5 instead of 1
  • decrement #count by 10 - Subtract 10
  • increment my value - Increment an element's value attribute
  • increment @data-score - Increment a data attribute

Reactive Updates:

The count mirror below the buttons uses: on change in #count to reactively update whenever the counter changes! This pattern watches for content changes in the #count element and automatically mirrors the value.
