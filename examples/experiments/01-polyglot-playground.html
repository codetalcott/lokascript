Grammar Transformations

How This Works

HyperFixi's grammar transformation system understands semantic roles (action, patient, event) rather than just keywords. When you write code in Japanese (SOV order: Object-Verb), it recognizes the pattern and can transform it to English (SVO order: Verb-Object) for execution. The demo elements above use real hyperscript via the _="" attribute - try clicking them, then try writing code to control them.
